# Passwords Generator (Test version)

A modern graphical desktop password generator built with Python using the tkinter and customtkinter libraries.

## Installation and Usage
To run the pre-compiled application on Windows, navigate to the project directory and double-click the executable file:
Passwords_generator_Test_version.exe

## Technical Details

The complete codebase has been thoroughly created, refactored, optimized, and verified by the author. All internal generation logic, text wrapping, input validations, and clipboard operations have been customized to ensure correct logical flow. The source code (Passwords_generator_Test_version.py) is fully open and available for modifications, redistribution, or code review.

IMPORTANT: any password that you created after the moment when new password was generated or when the application was closed it is permanently deleted from the program!
